# Olist SQL Project
Analysing Olist e-commerce data for business insights

The goal of this project is to use the Olist e-commerce dataset to extract valuable business insights.

## Olist E-commerce Dataset Overview
The Olist dataset is a comprehensive real-world dataset simulating e-commerce transactions. It provides rich details on customer interactions, products, orders, and reviews within Olist, a Brazilian e-commerce platform connecting sellers with buyers. This dataset is ideal for analysing the complexities of an online marketplace.
